# Nuno Neves Antiquorum — 3 propostas de site

Três versões completas, prontas a deploy, para escolha do cliente. Cada uma é um
projecto independente em **Next.js 15 · TypeScript · Tailwind 3 · GSAP**.

| Versão | Pasta | Conceito |
|--------|-------|----------|
| **V1** | [`v1-deco-noir`](v1-deco-noir) | Art Déco Noir — preto e dourado, dramático, fiel ao logo |
| **V2** | [`v2-galeria-leiloeira`](v2-galeria-leiloeira) | Galeria / leiloeira editorial — marfim e dourado, estilo casa de leilões |
| **V3** | [`v3-boutique-moderna`](v3-boutique-moderna) | Boutique moderna — verde e esmeralda, glassmorphism e micro-interacções |

## Correr localmente

```bash
cd v1-deco-noir   # ou v2-galeria-leiloeira / v3-boutique-moderna
npm install
npm run dev
```
